# Outdoor Squad — Thursday 2pm Call Run Sheet

Purpose: keep the call tight and convert Nicholas from interested/prototype mode into a paid 14-day pilot decision.

## 0. Call objective
End with one of:
1. **Paid pilot agreed** at $299 setup + optional $149/month continuation.
2. **Launch materials requested** with explicit agreement to start once received.
3. **Friday decision point** agreed if he needs to think.

Do not end on vague enthusiasm.

## 1. Opening — 30 seconds
> “I’ve kept this deliberately narrow: answer common enquiries, qualify good-fit leads, and push them toward the free intro / coach-call step. I’ll show the flow, ask a few scope questions, then if it feels useful we can decide whether to run a 14-day pilot.”

## 2. Demo path — 5 minutes
Run these in order:
1. “I’m interested but not very fit yet.”
2. “I work full-time and want evenings.”
3. “I’m not sure group training is for me.”
4. “Do you help with food?”
5. “My name is Sam, 0412 345 678.”

What to point out:
- It reassures beginners without sounding salesy.
- It qualifies schedule/goals before handoff.
- It captures contact details.
- It avoids overbuilding; first version is just the enquiry-to-intro path.

## 3. Discovery — 5 questions max
Ask only what affects launch:
1. “Where do most enquiries come from now — website, Instagram, phone, ads, referrals?”
2. “What’s the best next step: free intro, coach call, or a form/link?”
3. “What 3 questions should every lead answer before you follow up?”
4. “What objections stop people trying the first class?”
5. “After 14 days, what would make this worth keeping?”

## 4. Close — say this, then stop
> “Based on that, I’d keep the pilot narrow: website enquiries, qualification, contact capture, and the free intro / coach-call handoff. I can tune it around your answers and get version one live quickly. Want to run the 14-day pilot at $299, then only keep it at $149/month if it proves useful?”

## 5. If he hesitates
Use the exact resistance path:

### Needs to think
> “Totally fair. Should we set Friday as the decision point? I’ll send the short pilot summary, and if you send the Zoovr chat + preferred CTA before then I’ll tune the scope around your real flow.”

### Price resistance
> “We can make version one narrower: website FAQs + qualification + free intro handoff for $199. If it earns its keep, we expand later.”

### Wants more features
> “I’d avoid overbuilding before we know it works. Let’s prove the enquiry-to-intro path first, then add Instagram/WhatsApp/CRM once real conversations justify it.”

## 6. Immediate follow-up
After the call, send the matching draft from `POST-CALL-FOLLOW-UP-2026-04-29.md`:
- agreed live,
- thinking about it,
- narrow $199 starter.

## 7. Launch-material checklist
If he agrees or leans yes, get:
- preferred CTA link / destination,
- 3-5 qualification questions,
- lead notification email,
- Zoovr chat/example flow,
- any must-use wording or FAQs.
