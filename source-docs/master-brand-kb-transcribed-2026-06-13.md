## 3. Training Modalities & Class Types
Styles: kettlebell, strongman/odd-object (sandbags, sleds, deadballs, trap bars), CrossFit-style HIIT, bodyweight/animal flow. All classes scalable, all ages and levels.
- Strength'N'Tone: progressive resistance for functional strength/power; power bags, kettlebells, dumbbells, TRX, bodyweight, medicine balls, deadballs.
- HiiT'N'Run: HIIT for heart/lung capacity; circuits, sprints, speed/agility, hill/stair, animal movement.
- Flow'N'Flex: hybrid recovery/move-better session; blends strength+flexibility of YOGA with precise core control of PILATES + targeted restorative MOBILITY work; flowing sequences + strengthening; stretching to release desk tightness and post-training stiffness. (This is the UMBRELLA class — absorbed Yoga Squad + Power'N'Pilates.)
- Core'N'Sore: stabilise/endure/strengthen core, weighted + bodyweight + high-heart-rate finishers. PRIMARY HOME for animal flow work.
- Outdoor Hyrox: inspired by the Hyrox race; short running intervals + eight functional stations (push, pull, squat, hinge, lunge, carry, rotate, gait); sleds, power bags, kettlebells.
- Buff'N'Puff: hybrid Hyrox-style; resistance + cardio combined, time-optimised.
- YTP (Young'N'Strong): kids; physical literacy, athletic foundation, confidence, fun.
- SPT: 4-person max; personalised programming on 6-week cycles, 12-week assessment cadence; includes unlimited group classes.
NOTE: NO Power'N'Pilates as a separate class. NO Yoga Squad. Flow'N'Flex is the single umbrella.
